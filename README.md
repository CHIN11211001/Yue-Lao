# 🏮 月老 Yue Lao — 台灣人口統計視覺化分析

> 線上體驗網址：👉 **[https://CHIN11211001.github.io/Yue-Lao/](https://CHIN11211001.github.io/Yue-Lao/)**

歡迎來到 **「月老 Yue Lao」**，這是一個以現代網頁技術打造、專為台灣人口結構設計的**互動式數據視覺化平台**。

無論您是想了解理想伴侶在台灣的「稀有度」，還是單純對台灣的人口組成感到好奇，都能透過這個平台，進行多維度的交叉篩選，以最直觀的漏斗圖、雷達圖與長條圖，即時解析符合條件的群體佔比！

---

## ✨ 核心特色功能

- 🎯 **多維度條件篩選**：支援 11 項條件設定，包含：
  - 基本生理：性別、年齡、身高、體重
  - 社經地位：教育程度、月收入、產業別、居住區域
  - 狀態與性格：婚姻狀態、星座、MBTI 16 型人格
- 📊 **即時互動視覺化圖表**：
  - **條件覆蓋率雷達圖**：一秒看懂哪個條件過於嚴苛。
  - **動態長條圖**：各維度符合比例一目了然。
  - **無情漏斗分析**：層層篩選，計算真實出現的機率與推估人數。
- 🎨 **現代化流暢的 UI/UX 設計**：採用「玻璃態 (Glassmorphism)」設計語言搭配深色模式，帶來兼具專業與質感的數據分析體驗。
- ⚡ **純前端快速運算**：免後端伺服器，利用 React 狀態管理即時計算機率（基於獨立事件機率推估）。

## 📈 數據來源說明

本專案的基礎推估數據，整理自 2023-2024 年度台灣政府公開統計資料：
1. **內政部戶政司** (性別、年齡、婚姻狀況、區域分布)
2. **財政部 / 行政院主計總處** (受僱者收入分布、產業分布)
3. **衛生福利部國健署** (身高、體重分布推估)
4. **教育部** (教育程度分布)
5. **內政部統計處** [114年第25週內政統計通報：婚姻階梯](https://www.moi.gov.tw/News_Content.aspx?n=2905&s=328924) (年齡與教育程度交叉機率推估)
6. **行政院主計總處** [109年人口及住宅普查總報告提要分析](https://www.stat.gov.tw/News_Content.aspx?Create=1&n=2752&state=1327FD6AD8DCDA52&s=230649&ccms_cs=1&sms=11064) (全台真實物理人口常態機率分布，參考 p.23 表3-2 15歲以上常住人口之婚姻狀況)
7. **《我國2013-2016 年國民營養調查體位分析與常模建立-(I) 身高、體重、身體質量指數》** (年齡別身高、體重、BMI 常模；用於第一版 `P(身高 | 年齡, 性別)` 與 `P(體重 | 身高, 年齡, 性別)` 條件機率表)

*(註：星座與 MBTI 等趣味性數據為依據公開問卷與統計常態進行的大略分布推估，結果僅供參考與娛樂用途。本系統已對年齡、身高、體重、學歷、收入、產業、婚姻等部分維度加入條件機率修正，但真實世界條件仍具有更複雜的相依性。)*

## 🛠️ 開發技術棧

- **核心框架**：React 19 + TypeScript + Vite 6
- **狀態管理**：Zustand
- **圖表引擎**：Apache ECharts (`echarts-for-react`)
- **樣式設計**：純 CSS Variables + CSS Grid/Flexbox
- **部署方式**：GitHub Actions 自動部屬至 GitHub Pages

## 🚀 開發者指南 (本地執行)

如果您想在自己的電腦上執行或修改本專案，請按照以下步驟進行：

### 1. 安裝相依套件
請確保您的電腦已安裝 Node.js (建議版本 v18+)，然後在專案目錄下執行：
```bash
npm install
```

### 2. 啟動本機開發伺服器
```bash
npm run dev
```
啟動後，開啟瀏覽器前往 `http://localhost:5173/Yue-Lao/` 即可預覽專案。

### 3. 建置靜態檔案
```bash
npm run build
```
產出的檔案會位於 `dist/` 資料夾，可以直接用於靜態伺服器部署。
